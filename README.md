# dotfiles
my setting files.

- shell(fish or ~beef~ zsh)
- terminal(alacritty)
- tmux
- vim(neovim)
- powerline

## Usage
RECOMMEND: Makefile included. Do not copy all. Pickup you needed.

```
$ git clone https://github.com/oratake/dotfiles.git
$ cd dotfiles
$ make <PICKUP_YOU_NEEDED>
```
